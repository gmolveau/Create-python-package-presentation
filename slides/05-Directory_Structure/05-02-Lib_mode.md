## Lib mode

But with a more complex tool with **several python script files**, `.py` script files must be place under a "lib" folder.

```
└── tool-name
    ├── tool-name
    │   ├── __init__.py
    │   ├── script.py
    │   └── main.py
    ├── LICENCE
    ├── README.md
    └── setup.py
```

And of course a `__init__.py` will be needed.
